Breve descripción

- Proyecto web simple desarrollado como ejercicio/práctica.

- Estructura inicial del proyecto creada.
- Componentes principales añadidos (layout, páginas principales).
- Rutas básicas configuradas.
- Estilos básicos aplicados.
- Dependencias iniciales instaladas.

Cómo ejecutar (ejemplo)

- Instalar dependencias: npm install
- Ejecutar en modo desarrollo: npm start
- Construir para producción: npm run build

Estructura sugerida

- src/
  - components/ -> componentes reutilizables
  - pages/ -> vistas/páginas
  - services/ -> llamadas a API
  - styles/ -> estilos globales
  - assets/ -> imágenes y recursos
- public/ -> archivos públicos
